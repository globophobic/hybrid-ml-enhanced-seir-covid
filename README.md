# Hybrid Machine Learning Enhanced SEIR Model for Regional COVID-19

## Overview
This research project proposes an enhanced SEIR epidemiological model integrated with machine learning techniques to analyze and predict regional COVID-19 spread patterns.

## Key Contributions
- Improved classical SEIR compartmental model
- Integrated machine learning for prediction accuracy
- Regional COVID-19 data analysis
- Forecast visualization and comparison

## Repository Contents
- LaTeX source files (.tex)
- Final compiled research paper (PDF)
- Dataset used in modeling
- Simulation code / notebooks
- Graphs and results

## Tools Used
Python, Machine Learning, SEIR Mathematical Modeling, LaTeX, Data Analysis
